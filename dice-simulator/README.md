# Dice Roll Simulator

## Description
Simulates rolling two dice multiple times and counts how often each sum appears.
Prints both statistics and a histogram.

## Features
- Random dice rolls
- Counts frequency of each sum (2–12)
- Prints histogram with '*' for each occurrence
- User-controlled number of rolls

## How to Run
clang++ dice_simulator.cpp -o dice_simulator
./dice_simulator
