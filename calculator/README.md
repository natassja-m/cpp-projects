# C++ Calculator

## Overview
A command-line calculator built in C++ that performs basic arithmetic operations.

## Features
- Addition, subtraction, multiplication, and division
- User input handling
- Error handling for division by zero

## Technologies
- C++
- Command-line interface

## How to Run

Compile:
    clang++ calculator.cpp -o calculator

Run:
    ./calculator
